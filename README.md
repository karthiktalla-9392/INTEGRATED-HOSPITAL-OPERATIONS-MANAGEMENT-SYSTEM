# 🏥 AuraHealth Zenith | Next-Gen Wellness

AuraHealth Zenith is a sophisticated, responsive healthcare management dashboard designed to provide a "Premium Healthcare Intelligence" experience. It integrates medical inventory, live tele-consultation simulations, AI-driven symptom analysis, and automated diagnostic reporting.

## 🚀 Live Demo
**Your site is live at https://karthiktalla-9392.github.io/INTEGRATED-HOSPITAL-OPERATIONS-MANAGEMENT-SYSTEM/**

## ✨ Key Features
- **Smart Dashboard:** Real-time biometric trend visualization using **Chart.js**.
- **Zenith Pharmacy:** A dynamic inventory system with category-based filtering and cart management.
- **AI Symptom Engine:** Neural-mapping simulation for preliminary diagnosis based on user input.
- **Tele-Consultation:** Encrypted P2P video call simulation with specialist selection.
- **Diagnostic Vault:** Generates downloadable medical reports in PDF format using **jsPDF**.
- **Financial Node:** Integrated billing summary with UPI/QR code payment simulation.
